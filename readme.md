# Node Web Template

Serveur de base en nodejs.

## Installation et exécution:
```
$ npm install
$ npm start
```

## Quelques commandes clés:

bash:
```
$ env
$ export ENV_STAGE=dev
$ echo $ENV_STAGE
```

git:
```
$ git status
$ git add .
$ git commit -m "Message"
$ git log
$ git remote show
$ git push -u origin master
$ heroku create
$ git push heroku
```
